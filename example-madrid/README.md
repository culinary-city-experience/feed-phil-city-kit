# Madrid — reference example

This is the complete, working example the template was extracted from —
the actual guide built for a 27 Sep–4 Oct 2026 Madrid trip. Use it as the
model when filling in a new city's `city.js` and `city_geo.json`.

- `city.js` — all content: copy, venues, transit, links (see `../template/SCHEMA.md`)
- `city_geo.json` — all map geometry (streets, stations, landmarks, layout)
- `city.json`, `maps.json` — generated intermediate files (safe to delete;
  regenerated by `../builder/build.sh` or the manual steps in `PLAYBOOK.md`)
- `Feed-Phil-Madrid.pdf`, `Feed-Phil-Madrid.html` — the final deliverables

To rebuild from scratch:

```
../builder/build.sh .
```
